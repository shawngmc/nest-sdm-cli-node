# Usage
<!-- usage -->
```sh-session
$ npm install -g nest-sdm
$ nest-sdm COMMAND
running command...
$ nest-sdm (-v|--version|version)
nest-sdm/0.1.5 win32-x64 node-v12.0.0
$ nest-sdm --help [COMMAND]
USAGE
  $ nest-sdm COMMAND
...
```
